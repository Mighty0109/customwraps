모델 선택

Tesla Custom Wraps

모든 처리는 브라우저에서 로컬로 이루어집니다. 서버에 데이터가 전송되지 않습니다.
All processing happens locally in your browser.

Model 3

사용 방법

1. 이미지를 업로드하세요.
2. 캔버스에서 드래그하여 위치를 조정하세요.
3. 핀치 제스처로 크기와 회전을 조정하세요.
4. 슬라이더로 미세 조정하세요.
5. 내보내기 버튼으로 PNG를 다운로드하세요.

USB 설정

USB를 FAT32 또는 exFAT로 포맷하고, 루트에 /Wraps/ 폴더를 만든 후 PNG 파일을 넣으세요. Tesla에서 Toybox > Paint Shop을 열면 됩니다.

개인정보

이 앱은 서버에 어떤 데이터도 전송하지 않습니다. 모든 이미지 처리는 브라우저에서 로컬로 수행됩니다.
